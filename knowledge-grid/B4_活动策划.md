## 核心摘要

### 这是什么
活动策划是通过短期集中的运营动作，实现特定业务目标的系统性工作。

### 核心要点
1. **活动类型**：引流/促活/转化/品牌
2. **策划流程**：调研→创意→方案→执行→复盘
3. **核心指标**：参与人数/转化率/ROI

## 详细笔记

### 活动策划全流程
```
1. 调研阶段
   → 市场分析
   → 竞品研究
   → 用户洞察

2. 创意阶段
   → 主题构思
   → 形式设计
   → 创意提炼

3. 方案阶段
   → 目标拆解
   → 资源盘点
   → 排期规划

4. 执行阶段
   → 物料准备
   → 预热推广
   → 活动上线

5. 复盘阶段
   → 数据分析
   → 经验总结
   → 优化迭代
```

### 四类活动策划要点
```
引流型活动：
  → 目标：获取新用户
  → 特点：门槛低、诱饵强
  → 例：裂变活动、0元购

促活型活动：
  → 目标：激活沉睡用户
  → 特点：召回触达、专属福利
  → 例：唤醒活动、积分过期

转化型活动：
  → 目标：提升GMV
  → 特点：促销力度、限时限量
  → 例：双十一、秒杀

品牌型活动：
  → 目标：提升品牌认知
  → 特点：创意新颖、传播性强
  → 例：事件营销、联名活动
```
