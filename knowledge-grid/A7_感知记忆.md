## 核心摘要

### 这是什么
感知记忆是快速扫描和直觉反应的系统，能够在毫秒级别识别模式和触发响应。

### 核心要点
1. **快速扫描**：快速模式识别
2. **直觉反应**：基于经验的快速判断
3. **神经记忆**：模仿神经系统的记忆机制

## 详细笔记

### 感知记忆流程
```
输入信息
  ↓
快速模式识别（<10ms）
  ├─ 关键词匹配
  ├─ 语义相似度
  └─ 模式库检索

直觉反应触发（10-100ms）
  ├─ 常见模式 → 直接响应
  ├─ 模糊模式 → 进入分析
  └─ 新模式 → 创建记录

长期记忆关联（100ms+）
  ├─ 相关记忆检索
  ├─ 上下文关联
  └─ 知识网络匹配
```

### 感知优先级
```
P0 - 紧急/危险
  → 立即响应，不经过深思
  → 例：错误、异常、紧急请求

P1 - 高价值/高概率
  → 快速响应，简要分析
  → 例：常见问题、高频请求

P2 - 标准请求
  → 正常流程处理
  → 例：一般咨询、常规任务

P3 - 低优先级
  → 批量处理或延迟
  → 例：后台任务、探索性查询
```
