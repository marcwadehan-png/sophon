## 核心摘要

### 这是什么
数据运营是通过数据分析和指标监控，指导业务决策的运营方式。

### 核心要点
1. **指标体系**：北极星指标/过程指标/健康指标
2. **数据分析**：对比分析/漏斗分析/归因分析
3. **数据驱动**：用数据指导决策，用数据验证假设

## 详细笔记

### 常用数据分析方法
```
对比分析：
  → 时间对比（同比/环比）
  → 空间对比（不同渠道/品类）
  → 人群对比（不同用户群）

漏斗分析：
  → 转化路径识别
  → 流失点定位
  → 优化方向确定

归因分析：
  → 首次触点归因
  → 末次触点归因
  → 线性归因
  → 最终互动归因

相关分析：
  → 发现关联规律
  → 预测未来趋势
```

### 数据指标字典
```
用户指标：
  → DAU/WAU/MAU
  → 新增用户/流失用户
  → 留存率/活跃度

业务指标：
  → GMV/订单量/客单价
  → 转化率/复购率
  → 毛利率/净利率

流量指标：
  → PV/UV/访问时长
  → 跳出率/退出率
  → 新老访客占比
```
