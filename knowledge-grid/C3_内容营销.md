## 核心摘要

### 这是什么
内容营销是通过创作和分发有价值的内容，吸引和留住目标用户的营销方式。

### 核心要点
1. **内容策略**：选题规划/形式选择/渠道分发
2. **内容创作**：标题/正文/视觉
3. **效果评估**：阅读/互动/转化

## 详细笔记

### 内容营销漏斗
```
Awareness（认知）：
  → 品牌曝光
  → 内容：科普/娱乐/热点
  → 指标：曝光量/阅读量

Consideration（考虑）：
  → 价值认同
  → 内容：教程/评测/案例
  → 指标：互动率/分享率

Conversion（转化）：
  → 信任建立
  → 内容：促销/证言/对比
  → 指标：点击率/转化率
```

### 内容形式选择
```
图文类：
  → 公众号/小红书/知乎
  → 优势：信息密度高，便于深度阅读

短视频：
  → 抖音/快手/视频号
  → 优势：直观生动，传播性强

直播类：
  → 电商直播/知识直播
  → 优势：实时互动，即时反馈

音频类：
  → 播客/语音公众号
  → 优势：利用碎片时间，陪伴感强
```

### 内容创作技巧
```
标题公式：
  → 数字型：5个技巧让你的转化率翻倍
  → 疑问型：为什么你的文案总是不吸引人？
  → 对比型：月薪3000和30000的文案差在哪里？
  → 悬念型：这个运营技巧，90%的人都不知道

开头技巧：
  → 痛点切入
  → 故事引入
  → 数据吸引
```
