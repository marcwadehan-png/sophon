## 核心摘要

### 这是什么
策略运营是制定和执行业务策略的系统性工作，核心是找到增长路径并持续优化。

### 核心要点
1. **策略制定**：目标→路径→资源→执行
2. **数据分析**：指标监控/问题诊断/机会发现
3. **迭代优化**：假设→验证→调整→固化

## 详细笔记

### 策略运营框架
```
诊断阶段：
  → 业务现状分析
  → 问题识别
  → 机会发现

规划阶段：
  → 目标设定（SMART）
  → 策略路径设计
  → 资源需求评估
  → 时间排期

执行阶段：
  → 任务分解
  → 责任明确
  → 进度跟踪
  → 问题协调

复盘阶段：
  → 结果对比
  → 原因分析
  → 经验沉淀
  → 迭代优化
```

### 运营指标体系
```
北极星指标：
  → 核心业务目标
  → 全公司统一方向

过程指标：
  → 支撑北极星的中间指标
  → 例：转化漏斗各层

健康指标：
  → 业务健康度监测
  → 例：留存、活跃度
```
