## 核心摘要

### 这是什么
广告投放是在数字平台（如巨量引擎、腾讯广告、百度推广等）进行付费推广的运营工作。

### 核心要点
1. **投放平台**：巨量引擎/腾讯广告/百度/快手
2. **投放策略**：人群定向/出价策略/素材优化
3. **效果优化**：AB测试/数据分析/持续迭代

## 详细笔记

### 投放平台特点
```
巨量引擎（字节系）：
  → 抖音/头条/西瓜
  → 特点：算法精准，量级大
  → 适合：品牌曝光，效果转化

腾讯广告（微信系）：
  → 微信/QQ/腾讯新闻
  → 特点：社交数据，用户粘性强
  → 适合：品牌宣传，私域引流

百度推广：
  → 搜索/信息流
  → 特点：意图明确，转化率高
  → 适合：精准获客，效果投放

快手：
  → 快手APP
  → 特点：下沉市场，老铁文化
  → 适合：下沉市场，品牌种草
```

### 投放优化技巧
```
人群定向优化：
  → 基础属性：年龄/性别/地域
  → 行为特征：兴趣/搜索/消费
  → 相似人群：种子用户扩展

出价策略：
  → 手动出价：精细控制
  → 智能出价：自动优化
  → 阶梯出价：测试最优出价

素材优化：
  → AB测试不同素材
  → 关注CTR和CVR
  → 定期更新素材
```

### 投放指标
```
曝光指标：
  → 曝光量/CPM

点击指标：
  → 点击量/CTR/点击单价

转化指标：
  → 转化量/CVR/转化成本

综合指标：
  → ROI/ROAS
  → CPA/CPS
```

## 举一反三
- 广告投放 = 精准钓鱼
- 投放优化 = 持续AB测试
- ROI = 投放的生命线
