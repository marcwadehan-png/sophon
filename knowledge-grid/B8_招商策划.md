## 核心摘要

### 这是什么
招商策划是为商业项目引入品牌和商户的系统性工作，核心是构建商业生态。

### 核心要点
1. **品牌组合**：定位匹配/业态互补/层级合理
2. **招商策略**：目标画像/接触路径/谈判技巧
3. **运营支持**：政策支持/运营赋能/持续优化

## 详细笔记

### 招商工作流程
```
筹备阶段：
  → 市场调研
  → 业态规划
  → 租金方案

执行阶段：
  → 目标品牌梳理
  → 接触洽谈
  → 合同签署

开业阶段：
  → 装修跟进
  → 开业筹备
  → 营销联动

运营阶段：
  → 业绩追踪
  → 品牌调整
  → 租金优化
```

### 品牌层级模型
```
头部品牌（Anchor）：
  → 知名度高/客流强
  → 例：超市、影院

主力品牌（Major）：
  → 目标客群匹配
  → 例：服装、餐饮

特色品牌（Specialty）：
  → 差异化/新鲜感
  → 例：网红店、体验店

配套品牌（Amenity）：
  → 满足刚需
  → 例：便利店、药店
```

## 举一反三
- 招商 = 构建生态圈
- 品牌组合 = 用户体验设计
- 租金 = 价值交换
