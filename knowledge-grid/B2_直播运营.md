## 核心摘要

### 这是什么
直播运营是通过实时视频内容进行产品展示、互动和销售的一种运营方式。

### 核心要点
1. **三角定律**：主播 × 货品 × 场景 = 直播成功
2. **人货场**：人（主播能力）/货（选品策略）/场（场景氛围）
3. **数据指标**：观看/停留/互动/转化/GMV

## 详细笔记

### 直播三角定律
```
        主播
       /  |  \
      /   |   \
     /    |    \
    /     |     \
   /      |      \
  V       V       V
专业    选品    场景
可信    刚需    沉浸
亲和    价值   氛围
```

### 主播能力模型
```
核心能力：
1. 表现力 → 镜头感/感染力/情绪传递
2. 专业度 → 产品知识/行业洞察
3. 应变力 → 突发情况处理
4. 销售力 → 痛点挖掘/价值塑造
```

### 直播选品策略
```
引流款（限量/低价/引流）
  → 目的：吸引流量/聚集人气

利润款（核心/主推/稳定）
  → 目的：稳定收入/建立信任

形象款（高端/展示/提升）
  → 目的：提升品牌/展示实力
```

### 直播话术框架
```
1. 开场话术（0-3分钟）
   → 建立期待/抛出话题

2. 产品话术
   → FABE法则：
     - Feature（特征）
     - Advantage（优势）
     - Benefit（利益）
     - Evidence（证据）
     
3. 促单话术
   → 紧迫感/稀缺感/从众心理

4. 收尾话术
   → 下次预告/粉丝维护
```
