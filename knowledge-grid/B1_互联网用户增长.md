## 核心摘要

### 这是什么
用户增长是互联网运营的核心目标——通过系统化的方法让用户从少变多、从陌生变忠诚。

### 核心要点
1. **AARRR模型**：Acquisition→Activation→Retention→Revenue→Referral
2. **裂变思维**：通过已有用户带动新用户，实现指数级增长
3. **私域沉淀**：把用户从公域转到私域，降低获客成本

## 详细笔记

### AARRR漏斗详解
```
Acquisition 获取
  → 渠道：投放/内容/活动/SEO/裂变
  → 指标：CAC（获客成本）、渠道ROI

Activation 激活
  → 新用户首次体验到产品价值
  → 指标：激活率、首次体验时长

Retention 留存
  → 用户持续使用产品
  → 指标：次日/7日/30日留存率

Revenue 变现
  → 用户付费
  → 指标：ARPU、转化率、LTV

Referral 推荐
  → 用户主动分享传播
  → 指标：K因子（裂变系数）
```

### 裂变活动四要素
```
1. 海报设计
   → 视觉冲击力/价值感/紧迫感
   
2. 诱饵设置
   → 目标用户真正需要的东西
   → 虚拟资料/优惠券/会员权益
   
3. 分享机制
   → K因子 = 分享人数 × 转化率 > 1
   
4. 承接流程
   → 用户扫码后去哪里？
```

### 用户增长关键指标
| 指标 | 含义 | 优秀值 |
|------|------|--------|
| CAC | 获客成本 | 越低越好 |
| LTV | 用户生命周期价值 | LTV > 3×CAC |
| K因子 | 裂变系数 | ≥1 |
| 留存率 | 次日/7日/30日 | 行业对比 |
