## 核心摘要

### 这是什么
地产策划是房地产项目的营销策划工作，包括项目定位、案场运营、推广策略等。

### 核心要点
1. **项目定位**：市场调研/客群分析/差异化定位
2. **案场运营**：接待流程/氛围营造/逼定技巧
3. **推广策略**：线上/线下/渠道

## 详细笔记

### 地产项目定位
```
市场分析：
  → 区域市场容量
  → 竞品分析（户型/价格/去化）
  → 政策环境

客群画像：
  → 年龄/职业/家庭结构
  → 购房动机/关注因素
  → 支付能力

差异化定位：
  → 核心卖点提炼
  → 价值体系构建
  → 品牌调性确立
```

### 地产营销节点
```
筹备期：
  → 示范区开放
  → 蓄客积累

开盘期：
  → 集中成交
  → 氛围营造

续销期：
  → 持续推广
  → 促销政策

尾盘期：
  → 清盘策略
  → 资源整合
```

## 举一反三
- 地产策划 = 高净值用户运营
- 案场 = 线下体验店
- 开盘 = 电商大促
