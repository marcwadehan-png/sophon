## 核心摘要

### 这是什么
判断决策是整合多方意见、做出高质量决策的机制。借鉴"七人投票"和"太极阴阳"思想。

### 核心要点
1. **七人投票**：模拟多角色视角，确保决策全面
2. **太极阴阳**：平衡正反两面，找到动态平衡点
3. **决策树**：清晰展示决策路径和分支

## 详细笔记

### 七人投票模型
```
七个视角：
1. 用户视角 - 用户真正需要什么？
2. 业务视角 - 公司目标是什么？
3. 技术视角 - 能实现吗？
4. 财务视角 - 投入产出比？
5. 风险视角 - 潜在风险？
6. 竞争视角 - 市场格局？
7. 长期视角 - 3年后看现在？
```

### 太极决策法
```
任何决策都包含：
阴（保守面）←→ 阳（进取面）

决策质量 = 找到动态平衡点
- 太阴：过度保守，错失机会
- 太阳：过度冒进，引发风险
- 少阴：偏保守，稳健但可能保守
- 少阳：偏进取，活跃但可能冒进
- 太和：最佳平衡点
```

### 决策树示例
```
问题：是否上线新功能？

  ├─ 用户需求强？ ──┬─ 是 → 继续
  │               │
  │               └─ 否 → 终止
  │
  ├─ 技术可行？ ──┬─ 是 → 继续
  │               │
  │               └─ 否 → 优化或终止
  │
  ├─ ROI合理？ ──┬─ 是 → 继续
  │               │
  │               └─ 否 → 重新设计
  │
  └─ 风险可控？ ──┬─ 是 → 通过
                  │
                  └─ 否 → 降低风险后重评
```
