## 核心摘要

### 这是什么
广告策划是设计和投放商业广告的系统性工作，核心是找到目标用户并传递价值。

### 核心要点
1. **广告三要素**：创意/渠道/人群
2. **投放策略**：精准定向/预算分配/效果优化
3. **素材创意**：吸引力/相关性/转化力

## 详细笔记

### 广告投放漏斗
```
曝光（Impression）
  → 触达目标人群
  → 指标：曝光量/CPM

点击（Click）
  → 引起用户兴趣
  → 指标：点击量/CTR

转化（Conversion）
  → 用户完成目标行为
  → 指标：转化量/CVR

留存（Retention）
  → 用户成为长期客户
  → 指标：留存率/LTV
```

### 广告素材创意
```
高转化素材特征：
1. 吸引力：3秒内抓住眼球
2. 相关性：与目标用户痛点相关
3. 价值感：明确展示利益点
4. 紧迫感：限时/限量/专属
5. 信任感：权威/背书/案例
```

### 投放渠道选择
```
信息流广告：
  → 头条/抖音/微信
  → 特点：原生感、精准定向

搜索广告：
  → 百度/360/搜狗
  → 特点：意图明确、转化率高

展示广告：
  → banner/开屏/插屏
  → 特点：曝光强、品牌提升

KOL广告：
  → 达人/博主/主播
  → 特点：信任度高、种草能力强
```

## 举一反三

- 广告素材 = 第一印象管理
- 投放优化 = 持续AB测试
