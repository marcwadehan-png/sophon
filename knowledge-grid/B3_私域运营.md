## 核心摘要

### 这是什么
私域运营是将用户从公域沉淀到企业私有渠道，建立长期用户关系的运营方式。

### 核心要点
1. **私域价值**：用户资产私有化，降低获客成本
2. **三大载体**：企微/社群/公众号
3. **运营核心**：分层管理，精准触达

## 详细笔记

### 私域沉淀路径
```
公域（平台/广告）
  ↓ 投放/内容/活动
潜客
  ↓ 首次互动
私域（企微/社群/公众号）
  ↓ 价值提供
活跃用户
  ↓ 会员体系/专属权益
高价值用户
  ↓ 口碑传播
推荐/裂变
```

### 企微运营四步法
```
1. 引流
   → 公域转私域
   → 渠道：广告/内容/活动

2. 分层
   → 新用户 → 活跃用户 → 高价值用户
   → 标签管理，行为追踪

3. 运营
   → 内容推送（有价值）
   → 活动激活（有互动）
   → 专属服务（有温度）

4. 变现
   → 产品推荐
   → 会员转化
   → 裂变增长
```

### 社群运营SOP
```
入群前：
  → 明确社群定位
  → 设定入群门槛

入群时：
  → 欢迎语
  → 规则说明
  → 价值预告

入群后：
  → 内容运营（日常）
  → 活动运营（定期）
  → 情感运营（温度）

出群后：
  → 流失预警
  → 挽回机制
```

## 举一反三
- 私域 = 用户资产私有化
- 社群 = 线上俱乐部
- 企微 = 用户的"家"
