## 核心摘要

### 这是什么
反思进化是Somn持续自我改进的核心机制，通过ROI追踪和强化学习实现能力提升。

### 核心要点
1. **ROI追踪**：评估每次决策的效果
2. **强化学习**：从反馈中学习优化
3. **自我迭代**：不断改进决策模型

## 详细笔记

### ROI追踪体系
```
每次决策后追踪：
├─ 决策质量（是否有效解决问题）
├─ 执行效率（是否快速达成目标）
├─ 资源消耗（投入产出比）
└─ 长期影响（对未来的影响）

评估周期：
- 短期：决策后即时反馈
- 中期：24小时-7天
- 长期：30天以上
```

### 强化学习机制
```
强化信号：
+ 正向奖励：正确决策，好结果
- 负向奖励：错误决策，坏结果
0 中性信号：一般表现

更新策略：
1. 即时更新：小步快跑，快速迭代
2. 批量更新：积累经验，周期优化
3. 深度更新：重大决策，全面复盘
```

### 反思流程
```
1. 结果回顾
   → 实际结果 vs 预期结果

2. 原因分析
   → 决策正确？执行问题？外部因素？

3. 模式提取
   → 哪些决策倾向是有效的？

4. 模型更新
   → 调整决策权重，更新知识库

5. 经验归档
   → 重要经验存入长期记忆
```
