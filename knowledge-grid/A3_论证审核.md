## 核心摘要

### 这是什么
论证审核是决策输出前的最后一道关卡，通过翰林院三轮审核机制确保结论质量。

### 核心要点
1. **翰林院三轮审核**：逻辑论证 → 多视角反驳 → 综合评审
2. **四维反驳**：正/反/合/创
3. **防止偏颇**：识别认知偏差，确保客观性

## 详细笔记

### 翰林院三轮审核
```
第一轮：逻辑论证
  → 前提是否成立？
  → 推理是否有效？
  → 结论是否必然？

第二轮：多视角反驳
  → 正方视角：支持结论的证据？
  → 反方视角：反对结论的理由？
  → 中立视角：未考虑的因素？
  → 创意思维：突破框架的新角度？

第三轮：综合评审
  → 权衡各方观点
  → 给出qualified结论
  → 标注置信度和适用边界
```

### 四维反驳
- 正方支持：强化核心论点
- 反方质疑：识别潜在漏洞
- 中立审视：发现盲点假设
- 创意突破：重构问题框架
