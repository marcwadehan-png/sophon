## 核心摘要

### 这是什么
电商运营是在电商平台（如淘宝、京东、拼多多等）进行产品销售和店铺管理的运营工作。

### 核心要点
1. **人货场**：消费者/商品/购物场景
2. **流量运营**：免费/付费/私域
3. **转化优化**：主图/详情/评价/客服

## 详细笔记

### 电商运营核心
```
流量端：
  → 搜索流量（SEO/标题优化）
  → 推荐流量（算法优化）
  → 付费流量（推广投放）
  → 私域流量（会员/社群）

转化端：
  → 主图视频（吸引力）
  → 详情页（说服力）
  → 评价管理（信任感）
  → 客服咨询（转化率）

供应链端：
  → 选品策略
  → 库存管理
  → 物流体验
```
