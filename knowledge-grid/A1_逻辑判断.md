## 核心摘要

### 这是什么
逻辑判断是思考问题时的"推理引擎"。它不是拍脑袋，而是有结构、有步骤的思考过程。

### 核心要点
1. **三层结构**：前提（Premise）→ 推理（Inference）→ 结论（Evaluation）
2. **四种推理模式**：演绎（必然）/ 归纳（概然）/ 类比（迁移）/ 反事实（假设）
3. **谬误检测**：识别8种常见逻辑谬误
4. **来源**：翰林院审核模块

## 详细笔记

### 三层推理结构
```
第一层：前提识别
  → 事实是什么？假设是什么？
  → 信息来源可靠吗？
  
第二层：推理过程
  → 从前提能推出什么？
  → 推理模式是否正确？
  
第三层：结论评估
  → 结论是否从前提必然/合理推出？
  → 有没有反例？
```

### 谬误检测清单
- 人身攻击（Ad Hominem）：攻击人不攻击观点
- 稻草人谬误：曲解对方观点再反驳
- 滑坡论证：无必然联系的递推
- 循环论证：用结论证明前提
- 诉诸权威：权威≠真相
- 诉诸情感：情感≠事实
- 虚假两难：非此即彼陷阱
- 以偏概全：样本偏差

## 思维模型
**逻辑判断决策流程：**
```
接收问题/信息
  ↓
前提提取（事实/假设分离）
  ↓
推理模式选择（演绎/归纳/类比/反事实）
  ↓
谬误检测（8项检查）
  ↓
结论生成
  ↓
输出（有效结论 / 需补充前提 / 结论不成立）
```
